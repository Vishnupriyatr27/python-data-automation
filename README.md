# Smart Data Processing & Automation System

## Overview
This project is a Python-based data processing and automation system that performs data validation, cleaning, and storage using SQLite.

## Features
- CSV data ingestion
- Data validation (age, email format, missing fields)
- Automated data processing
- SQLite database integration
- CRUD operations

## Tech Stack
- Python
- SQLite

## How to Run
1. Clone the repository
2. Run: python main.py

## Use Case
This project simulates real-world ETL pipelines used in backend systems and data processing applications.
